**功能用途：** 提供鄉鎮市區的詳細預報（未來3天連總效要本模故空…、未來1週連日），包含温度、湾度、風向、風速等。

**可自訂參數：**
- `locationName`：縣市/ 鄉鎮名稱
- `elementName`：氣象要素（例：T、RH）
- `timeFrom`、`timeTo`：時間區間
- `format`：JSON 或 XML

**官方連結：** [中央氣象署開放資料](https://opendata.cwa.gov.tw/dataset/forecast/F-D0047-093)

### 對應 API URL
[預報 GET /v1/rest/datastore/F-D0047-093](https://opendata.cwa.gov.tw/dist/opendata-swagger.html?urls.primaryName=openAPI#/%E9%A0%90%E5%A0%B1/get_v1_rest_datastore_F_D0047_093)

### 回傳資料欄位（常見）
- `datasetDescription`：資料集描述。
- `locationName`：縣市/鄉鎮名稱。
- `elementName`：氣象要素（例：`T` 溫度、`RH` 濕度、`Wx` 天氣現象、`PoP` 降雨機率、`WS` 風速、`WD` 風向、`CI` 體感/舒適度等）。
- `startTime`、`endTime` 或 `time`：預報時間（3小時步進與逐日）。
- `parameterName`：要素中文描述。
- `parameterValue`：數值或代碼值。
- `parameterUnit`：單位（°C、%、mm、m/s 等）。
